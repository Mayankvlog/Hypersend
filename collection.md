# Hypersend Database Collections

## Core User Collections
1. users - User profiles and authentication data
2. refresh_tokens - JWT refresh tokens for sessions
3. reset_tokens - Password reset tokens

## Chat & Messaging Collections  
4. chats - Chat rooms and group information
5. messages - Chat messages and metadata
6. group_activity - Group activity logs and events

## File & Media Collections
7. files - File metadata and information
8. uploads - Upload session tracking
9. media - Media file metadata

## Device & Security Collections
10. device_keys - Device key mappings
11. pre_keys - Pre-key bundles for encryption
12. signed_pre_keys - Signed pre-keys
13. identity_keys - Identity key pairs
14. sessions - Encryption sessions
15. sender_keys - Sender key distributions

## Relationship & Social Collections
16. contacts - User contact relationships
17. blocked_users - Blocked user relationships
18. relationship_graph - Social graph connections

## Cache & Temporary Collections
19. user_cache - Cached user data
20. chat_cache - Cached chat data
21. message_cache - Cached message data
22. presence_cache - User presence status
23. rate_limits - API rate limiting data

## Analytics & Monitoring Collections
24. user_analytics - User behavior analytics
25. chat_analytics - Chat usage statistics
26. file_analytics - File transfer analytics
27. system_metrics - System performance metrics

## Configuration Collections
28. app_config - Application configuration
29. feature_flags - Feature toggle settings
30. system_settings - System-wide settings

## Backup & Archive Collections
31. archived_messages - Archived message data
32. deleted_files - Deleted file metadata
33. audit_logs - System audit trail
34. backup_metadata - Backup tracking information
