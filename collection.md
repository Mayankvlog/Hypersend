# Hypersend Database Collections

## Core Collections
1. users - User profiles and authentication
2. refresh_tokens - JWT refresh tokens
3. reset_tokens - Password reset tokens

## Chat Collections
4. chats - Chat rooms and groups
5. messages - Chat messages
6. group_activity - Group activity logs

## File Collections
7. files - File metadata
8. uploads - Upload sessions
9. media - Media files

## Security Collections
10. device_keys - Device encryption keys
11. pre_keys - Pre-key bundles
12. signed_pre_keys - Signed pre-keys
13. identity_keys - Identity key pairs
14. sessions - Encryption sessions

## Social Collections
15. contacts - User contacts
16. blocked_users - Blocked users
17. relationship_graph - Social connections

## Cache Collections
18. user_cache - User data cache
19. chat_cache - Chat data cache
20. message_cache - Message cache
21. presence_cache - User presence
22. rate_limits - Rate limiting

## System Collections
23. app_config - Application settings
24. audit_logs - System audit trail
